# Admin Pages

This folder contains all ADMIN/BACKEND pages that use ODL components and design system.

## Rules:
- MUST use ODL components
- MUST use ODLTheme for styling
- Follow ODL patterns from Build project
- Internal admin interface

## Example:
```tsx
// ODL components only
import Button from '../../components/Button/Button';
import ODLTheme from '../../styles/ODLTheme';

<Button variant="primary">Admin Action</Button>
```
